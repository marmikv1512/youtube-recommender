
 AI YouTube Video Recommender System

This project is an AI-powered recommendation engine that suggests similar YouTube videos based on video descriptions and titles.

 Features
- Uses **YouTube Data API** to collect real video metadata  
- Applies **Sentence Transformers (MiniLM)** to generate embeddings  
- Uses **Cosine Similarity** for ranking similar videos  
- Clean and interactive **Streamlit UI**  
- End-to-end ML pipeline: data → embedding → recommendation → deployment  

 Tech Stack
- Python  
- Google YouTube Data API  
- Sentence Transformers  
- sklearn  
- Streamlit  
- Pandas / NumPy
